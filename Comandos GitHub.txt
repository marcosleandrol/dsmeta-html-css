Criar um novo repositório na linha de comando

1 - git init 
2 - git add .
3 - git commit -m "Mensagem explicativa"
4 - git branch -M main
5 - git remote add origin git@github.com:marcosleandrol/dsmeta-html-css.git (copiar ao criar novo repositorio)
6 - git push -u origin main
