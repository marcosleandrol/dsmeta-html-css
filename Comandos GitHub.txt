Criar um novo repositório na linha de comando

echo "# dsmeta-html-css" >> README.md 
git init 
git add README.md 
git commit -m "first commit" 
git branch -M main 
git remote add origin git@github.com:marcosleandrol/dsmeta-html-css.git
git push -u origin main

Enviar um repositório existente a partir da linha de comando

git remote add origin git@github.com:marcosleandrol/dsmeta-html-css.git
 git branch -M main 
git push -u origin main
